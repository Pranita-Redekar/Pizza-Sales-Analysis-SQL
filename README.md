<h1 align="center">🍕 Pizza Sales Analysis </h1>

<p align="center">
A data analysis project using SQL to uncover insights from pizza sales data.
</p>

<hr>

<h2>📌 Project Overview</h2>
<p>
This project focuses on analyzing pizza sales data using SQL. 
The goal is to extract meaningful insights such as revenue trends, customer preferences, and order patterns.
</p>

<h2>🛠️ Tools & Technologies Used</h2>
<ul>
  <li>SQL (PostgreSQL)</li>
  <li>Joins</li>
  <li>Aggregate Functions</li>
  <li>Window Functions</li>
  <li>Group By</li>
</ul>

<h2>📈 Key Analysis Performed</h2>
<ul>
  <li>Total number of orders placed</li>
  <li>Total revenue generated</li>
  <li>Highest-priced pizza</li>
  <li>Most common pizza size ordered</li>
  <li>Top 5 most ordered pizzas</li>
  <li>Category-wise sales analysis</li>
  <li>Hourly order distribution</li>
  <li>Average pizzas ordered per day</li>
  <li>Top pizzas based on revenue</li>
  <li>Revenue contribution by category</li>
  <li>Cumulative revenue over time</li>
  <li>Top pizzas per category using window functions</li>
</ul>

<h2>📊 Key Insights</h2>
<ul>
  <li>Identified the most popular pizza types and sizes among customers</li>
  <li>Discovered peak order hours during the day</li>
  <li>Analyzed revenue contribution by different categories</li>
  <li>Found high-performing pizzas driving maximum revenue</li>
</ul>
